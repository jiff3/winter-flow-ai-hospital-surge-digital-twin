# WINTER-Flow 2-Minute Demo Script

Use this flow for a quick portfolio or interview demo.

1. Launch the app:

   ```bash
   streamlit run app.py
   ```

2. In the sidebar, select `Severe Combined Surge`.

3. Open the `Command Center` tab and run the current simulation.

4. Show the red or amber risk labels, peak trolley count, ward occupancy, ICU occupancy, staff stress, and ED wait KPIs.

5. Open the `Policy Sandbox` tab.

6. Apply a package such as surge beds, extra staff, and discharge acceleration.

7. Run the before/after comparison and point out improvements in trolley count, wait time, and risk score.

8. Open the `Optimizer` tab and review the recommended plan table and projected score reduction.

9. Open the `Report` tab.

10. Preview the executive report and download the Markdown report plus CSV extracts.

## Demo Framing

WINTER-Flow uses synthetic data by default and is intended as a portfolio/research demonstration. It is not a clinical decision-support tool and should not be used for real hospital operations.
